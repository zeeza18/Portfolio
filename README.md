# Mohammed Azeezulla · Netflix-Inspired Portfolio

Personal portfolio showcasing agentic automation, multimodal analytics, and production-grade ML workstreams.

## Getting Started

1. Install dependencies: 
pm install
2. Create a .env with any REACT_APP_DATOCMS_* tokens if you want live CMS data.
3. Development server: 
pm start
4. Production build: 
pm run build

## Tech Stack
- React 18 + TypeScript
- Custom Netflix-inspired design system (glassmorphism, gradients, motion)
- GraphQL with graceful fallbacks when DatoCMS credentials are absent
- Animated intro sequence with audio cue
